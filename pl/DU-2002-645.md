# Obwieszczenie Prezesa Rady Ministrów z dnia 30 kwietnia 2002 r. o sprostowaniu błędów

## Treść obwieszczenia

Na podstawie art. 17 ust. 3 ustawy z dnia 20 lipca 2000 r. o ogłaszaniu aktów normatywnych i niektórych innych aktów prawnych w rozporządzeniu Rady Ministrów z dnia 19 października 2001 r. w sprawie bezpieczeństwa i znakowania produktów włókienniczych prostuje się następujące błędy w załączniku nr 11:

1) w pkt I.8.1.1. Wariant 1 zamiast wzoru:

> \( P_2 = \left[ {{{d_4 } \over {d_3 }} - d_4 \cdot {{r_2 } \over {m_3 }} + {{r_1 } \over {m_1 }} \cdot \left( {1 - {{d_4 } \over {d_3 }}} \right)} \right] \cdot 100\% \)

powinien być wzór:

> \( P_2 = \left[ {{{d_4 } \over {d_3 }} - d_4 \cdot {{r_2 } \over {m_2 }} + {{r_1 } \over {m_1 }} \cdot \left( {1 - {{d_4 } \over {d_3 }}} \right)} \right] \cdot 100\% \)

2) w pkt I.8.2 zamiast wzoru:

> \( A = 1 + {{a_1 - b_1 } \over {100}} \)

powinien być wzór:

> \( A = 1 + {{a_1 + b_1 } \over {100}} \)
